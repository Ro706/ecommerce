## Running the Application

1. **Clone the repository:**

   ```bash
   git clone  https://github.com/Ro706/ecommerce.git 
   cd ecommerce
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the application:**

   ```bash
   npm start
   ```

4. **Run tests:**

   ```bash
   npm test
   ```
